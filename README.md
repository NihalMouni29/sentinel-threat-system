# Sentinel Threat System

##  Overview

Sentinel Threat System is a cybersecurity application designed to detect insider threats using behavioral analysis and threat detection techniques.

---

##  Problem

Insider threats like data leaks and unauthorized access are hard to detect with traditional systems.

---

##  Solution

This system analyzes user activity logs to identify suspicious patterns and flag potential threats.

---

##  Features

* Behavioral activity analysis
* Threat detection engine
* Log processing
* Simple web interface (Flask)

---

##  Tech Stack

* Python
* Flask
* HTML/CSS
* CSV Data

---

##  Run Locally

```bash
git clone https://github.com/NihalMouni29/sentinel-threat-system.git
cd sentinel-threat-system/insider-threat
pip install -r requirements.txt
python app.py
```
